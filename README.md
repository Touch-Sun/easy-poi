直接Git clone; 干！

1. 准备好IDEA
2. JDK1.8

###### **安装步骤**

1. 下载本项目提供的Jar
2. 添加为程序的依赖库文件

```sh
git clone https://github.com/Touch-Sun/easy-poi.git
```
